Framework Integration:

You can use Bootstrap components directly with frameworks like React, Angular, or Vue.js. Here's an example with React: 

      
        import 'bootstrap/dist/css/bootstrap.min.css';
        function App() {
          return <button className="btn btn-primary">Click Me</button>;
        }
